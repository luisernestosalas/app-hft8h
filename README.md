# CRM con Clientes y Notas

Una aplicación web simple y eficiente para la gestión de clientes y sus notas asociadas, desarrollada con HTML y JavaScript vanilla.

## 🚀 Demo

Puedes ver la aplicación en funcionamiento aquí: [CRM Live Demo](https://app-hft8h-58q7iy6a8-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- **Gestión de Clientes**: Crear, editar, eliminar y visualizar clientes
- **Sistema de Notas**: Añadir notas específicas para cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador
- **Responsive**: Compatible con dispositivos móviles y desktop

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura de la aplicación
- **JavaScript (Vanilla)**: Lógica de negocio y manipulación del DOM
- **CSS3**: Estilos y diseño responsive
- **LocalStorage**: Persistencia de datos en el cliente
